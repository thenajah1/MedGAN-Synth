# MedGAN-Synth
MedGAN-Synth est un projet de génération de dossiers médicaux synthétiques à l’aide de réseaux antagonistes génératifs (GAN). Ce système permet de créer des données de santé réalistes tout en préservant la confidentialité des patients, facilitant ainsi la recherche, le développement d’algorithmes et la formation sans exposer de données sensibles.
Le projet combine un prétraitement avancé des données réelles, une architecture GAN adaptée aux données mixtes (numériques et catégorielles), et une génération contrôlée pour produire des dossiers médicaux synthétiques exploitables
